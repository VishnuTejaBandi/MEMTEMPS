## Getting Started <a  name = "getting_started"></a>

Install these first

- node **v14.18.1**
- vscode
- python **3.10.4**
- Add Eslint and Prettier Extensions in vscode

Clone this project.

## Getting the app Running[DEV] <a  name = "install"></a>

**Client**

Open client folder in terminal and do -

```
npm install
npm run serve
```

**Server**

```
python main.py
```
